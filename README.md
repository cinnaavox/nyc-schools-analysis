# NYC Schools Data Analysis

This project explores multiple datasets related to NYC high schools, including school directories, demographics, safety reports, and SAT results.

The goal of the project was to practice real-world data workflows including:

- data cleaning
- exploratory data analysis
- SQL queries executed via Python
- database population
- data validation and schema design

The work was completed as part of a data onboarding project.

## Project Structure

incident_analysis  
Exploration of safety incidents in NYC schools.

school_directory_exploration  
Exploratory analysis of the NYC High School Directory dataset.

database_queries  
SQL queries executed via Python to analyze school data.

database_population  
Cleaning and inserting SAT results data into a PostgreSQL database.

Kaggle – NYC SAT Results Dataset
