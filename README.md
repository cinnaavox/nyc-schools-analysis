# 📊 NYC Schools Data Analysis

This project explores multiple datasets related to NYC high schools, including school directories, demographics, safety reports, and SAT results.

The goal of this project was to practice real-world data workflows including:

- 🧹 **Data cleaning**
- 🔎 **Exploratory data analysis (EDA)**
- 🧠 **SQL queries executed via Python**
- 🗄️ **Database population**
- 📐 **Data validation and schema design**

The work was completed as part of a **data onboarding project**.

---

# 📂 Project Structure

## 🚨 incident_analysis
Exploration of safety incidents reported in NYC schools.

## 🏫 school_directory_exploration
Exploratory analysis of the NYC High School Directory dataset using Python and Pandas.

## 🗃️ database_queries
SQL queries executed via Python to analyze NYC school data across multiple tables.

## 🧮 database_population
Cleaning SAT results data and inserting the cleaned dataset into a PostgreSQL database.
